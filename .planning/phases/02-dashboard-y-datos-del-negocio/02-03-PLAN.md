<objective>
Construir la capa de autenticación y aislamiento del dashboard: los tres clientes Supabase (browser anon, server anon+cookies, admin service_role server-only), el middleware que refresca sesión y gatea rol, el helper require-role, el login/logout, y el script de verificación de aislamiento cross-tenant. Purpose: Sin esta capa nadie puede loguearse ni existe el gate owner/superadmin; es prerequisito de todo el CRUD y del panel superadmin.
Output: clientes, middleware, login page, auth actions, require-role, verify-dashboard-isolation.ts.
</objective>
